Modern Web Music Player - Documentation

Project Overview

This is a comprehensive web-based music player application built as part of the Unified Mentor internship program. The project demonstrates modern web development practices and provides a rich user experience for playing audio files.

[Music Player Interface Screenshot]

Key Features

Audio Playback

Full control over audio files with play, pause, seek, and volume controls using HTML5 Audio API.

Playlist Management

Dynamic playlist with add, remove, and reorder functionality. Supports multiple audio formats.

Advanced Controls

Shuffle, repeat modes, and keyboard shortcuts for enhanced user experience.

Responsive Design

Works seamlessly across all devices - desktop, tablet, and mobile with adaptive layouts.

Theme Support

Light and dark theme options with persistent user preferences.

File Upload

Drag & drop functionality for easy file uploads with support for multiple formats.

Technology Stack

HTML5 CSS3 JavaScript ES6+ Web Audio API Service Worker LocalStorage Font Awesome

Installation & Usage

1. Clone or download the project files 2. Open index.html in a modern web browser 3. For file upload functionality, serve files using a local server: # Using Python python -m http.server 8000 # Using Node.js npx http-server # Using VS Code Live Server Right-click index.html → "Open with Live Server"

Code Architecture

JavaScript Classes & Functions

  • MusicPlayer Class: Main application class handling all functionality
  • Audio Management: Play, pause, seek, volume control
  • Playlist System: Add, remove, shuffle, repeat functionality
  • UI Management: Theme switching, responsive controls
  • File Handling: Upload, drag & drop, format validation
  • Event Listeners: User interactions and audio events

CSS Features

  • CSS Grid and Flexbox for responsive layouts
  • CSS Custom Properties for theme management
  • Smooth animations and transitions
  • Media queries for mobile optimization
  • Modern gradients and visual effects

Browser Compatibility

  • Edge 79+

Supported Audio Formats

  • MP3 (.mp3)
  • WAV (.wav)
  • OGG (.ogg)
  • M4A (.m4a)

Performance Features
